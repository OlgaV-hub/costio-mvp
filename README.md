# Costio MVP

Costio es una calculadora de costos para pequeños emprendedores gastronómicos.

El objetivo del proyecto es ayudar a los usuarios a calcular el costo real de sus productos, definir precios de referencia, gestionar pedidos y visualizar la rentabilidad del negocio.

## Alcance del MVP

La primera versión contempla:

- registro e inicio de sesión con Google;
- gestión de productos, categorías e ingredientes;
- cálculo de recetas y costos de producción;
- cálculo de costo por unidad útil y por presentación de venta;
- simulación de precio de venta y margen;
- recepción de pedidos desde WhatsApp Business;
- gestión de pedidos y snapshot económico;
- dashboard general con indicadores de ingresos, costos, ganancia y margen.
